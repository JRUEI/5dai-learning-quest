# 5DAI Learning Quest

Day 1 的個人學習資料庫與進度儀表板。這是純前端網站，進度與筆記保存在瀏覽器的 `localStorage`。

## 本機使用

直接開啟 `index.html` 即可。若瀏覽器限制本機檔案行為，可在資料夾內啟動任意靜態伺服器。

## 資料與隱私

公開 repository 僅包含自行撰寫的學習追蹤介面與摘要式項目名稱，沒有上傳課程原始 PDF、完整 Podcast 字幕或作業原文。原始教材請保留於本機。

## 擴充 Day 2-5

目前 Day 1 資料位於 `app.js` 的 `items` 與 `slides` 陣列。後續可將資料抽成 `data/day-*.js`，並為每一天建立對應頁面；`localStorage` 的資料格式已可延續使用。
